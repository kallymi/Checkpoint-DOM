Frequently asked questions

Is there a Figma version for Flowbite Icons?

You can start using the full collection of SVG icons in Figma by checking out and downloading the official Flowbite Icons Figma file from the community page and start using them in your web application designs.

Can I use the SVG icons for free in my project?

Flowbite Icons is free and open-source under the MIT license which means you can use the collection of icons in your personal and commercial projects as long as you give credits to the author in the credits section (in this case being Flowbite).

Can I use the icons commercially?

You can use the SVG icons in a commercial project and you can consider giving proper credits to the author (Flowbite) and link back to the Flowbite website. Mentioning Flowbite is mandatory if the commercial product is a theme, templates, UI kit, UI component library or anything that can be considered as a direct competitor to Flowbite.

Can I use the icons in website templates and sell them?

You are allowed to use Flowbite Icons to build website templates and themes for sale or for free. In this case, you must include a link to the Flowbite Icons page in a prominent way at the credits or about section.

Do I have to mention Flowbite when using the icons?

If you are using Flowbite Icons in a personal or commercial project that is not a theme, template, UI kit, UI component library or anything that could be considered a direct competitor to Flowbite, you do not have to mention Flowbite unless you want to support the project.

Can I use the icons with React (JSX) or Vue?

To start using Flowbite Icons with React (JSX format) you can select the copy as option for JSX to directly copy and paste the JSX version of the icons that is compatible with the React source files.

What is the best UI library to use with Flowbite Icons?

This collection of SVG icons has been crafted for custom usage with the Flowbite Library which includes a core set of UI components, hundreds of blocks and templates and is best used with Tailwind CSS.

However, you can use the icons with any other UI library or front-end library of your choice as we also give you the possibility to export the raw SVG code.

How can I support the development of this project?

We built Flowbite Icons as a free, accessible and open-source package of SVG icons that you can use for personal and commercial projects and we will keep it that way.

However, developing these icons takes a lot of time and resources. If you want to support the development of this project, you can consider purchasing the pro version of Flowbite that includes hundreds of advanced UI componnents, blocks, templates, and Figma design system that you can leverage to build websites even faster.

Where can I get help and support?

There is no direct customer support when using Flowbite Icons (unless you have a pro license), however, you can submit an issue on GitHub or tune in with our community on Discord and ask for help there.
